# MISSÃO: Criação do Guia de Estilo e Tema para o Projeto Simpix

**PERSONA:** Você é um especialista em Design Systems e UI/UX, com profundo conhecimento em `Tailwind CSS` e na biblioteca de componentes `shadcn/ui`.

**CONTEXTO:** A estrutura funcional do nosso software `simpix` está completa. A base de componentes é `shadcn/ui`. Agora, precisamos refinar todo o design para que ele seja moderno, tecnológico e coeso. A identidade visual da marca `simpix` utiliza a fonte "Poppins", a cor primária "azul marinho" e uma cor de destaque "verde fluorescente".

**AÇÃO:** Sua missão é criar um **"Guia de Estilo e Tema"** completo para o `simpix`. Este guia será a nossa "fonte da verdade" para todo o design e servirá de base para um prompt futuro que irá instruir o `Agente Replit` a aplicar este novo design em todo o software.

**ETAPAS:**

1.  **Definição do Tema (`tailwind.config.ts`):**
    * Forneça a configuração de tema completa para nosso arquivo `tailwind.config.ts`.
    * Defina o `azul marinho` como a cor primária (`primary`) e o `verde fluorescente` como a cor de destaque (`accent`).
    * Sugira cores complementares para fundo (`background`), texto (`foreground`), cards, inputs, etc., que harmonizem com as cores principais, seguindo um padrão moderno (modo escuro e claro).

2.  **Configuração da Fonte (Poppins):**
    * Detalhe o passo a passo de como importar a fonte "Poppins" do Google Fonts e aplicá-la como a fonte padrão em todo o projeto através do arquivo CSS global.

3.  **Diretrizes de Componentes (`shadcn/ui`):**
    * Forneça diretrizes de como devemos aplicar este novo tema aos nossos componentes `shadcn/ui` existentes.
    * Exemplo: "O `Button` primário deve usar a cor `primary` como fundo. O `Button` secundário deve ter um fundo `secondary` com texto `primary`."

4.  **O "Prompt Ideal" para o `Agente`:**
    * Finalmente, com base em todo o guia que você criou, formule o **"Prompt Mestre de Design"** ideal. Este prompt deve ser o comando final que daremos ao `Agente Replit` para que ele refatore todo o frontend da aplicação, aplicando o novo tema, a nova fonte e as diretrizes de componentes.
