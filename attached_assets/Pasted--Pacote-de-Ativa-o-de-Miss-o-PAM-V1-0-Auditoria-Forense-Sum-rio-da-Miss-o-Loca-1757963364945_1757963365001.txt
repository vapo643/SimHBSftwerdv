* **Intenção Estratégica:** Diagnosticar precisamente por que um valor de `300000` armazenado corretamente no banco de dados está a ser exibido como `R$ 300` na interface do analista. A correção desta falha é crítica para a integridade das decisões de crédito.
* **Modelo Mental (Como se Encaixa):** O fluxo é: API envia um JSON com o campo `rendaMensal` (tipo `numeric`). Antes de renderizar na tela, uma função utilitária de formatação é chamada para converter o número em uma string `R$ xxx,xx`. A falha está, muito provavelmente, dentro desta função de formatação.
