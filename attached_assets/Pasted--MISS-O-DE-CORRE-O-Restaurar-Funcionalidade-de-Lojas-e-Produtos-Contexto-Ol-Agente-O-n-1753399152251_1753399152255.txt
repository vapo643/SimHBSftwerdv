# MISSÃO DE CORREÇÃO: Restaurar Funcionalidade de Lojas e Produtos

**Contexto:** Olá, Agente. O nosso painel de administração está com duas funcionalidades críticas quebradas que precisam da sua atenção. A nossa última colaboração para corrigir as `roles` foi um sucesso, então vamos seguir a mesma linha para resolver estes problemas.

**Lista de Bugs a Corrigir:**

* **1. O Módulo de Produtos está Completamente Quebrado:**
    * **O Problema (Sintoma):** Quando eu, como Administrador, entro na página de "Gestão de Produtos", a lista de produtos que existia simplesmente sumiu. Pior, ao tentar criar um novo produto no formulário, o frontend mostra um erro `response.json is not a function`.
    * **O Diagnóstico Provável:** Este erro de `response.json` quase sempre significa que a API de backend, neste caso `GET /api/produtos`, está a crashar e a retornar uma página de erro em HTML em vez do JSON que o frontend espera.
    * **Sua Tarefa:** Investigue a rota `GET /api/produtos` no nosso backend. Encontre e corrija o bug que está a causar o crash do servidor. O objetivo é que a API volte a retornar a lista de produtos em formato JSON de forma estável.

* **2. O Formulário de Lojas não Carrega os Parceiros:**
    * **O Problema (Sintoma):** Na tela de "Gestão de Lojas", a listagem funciona, mas ao clicar em "Nova Loja" ou "Editar", o formulário abre, mas o campo para selecionar o "Parceiro" não carrega nenhum dado, ficando vazio. Isto torna impossível criar ou editar uma loja, pois não consigo associá-la a um parceiro.
    * **O Diagnóstico Provável:** Isto é um bug de data fetching no frontend. O componente do formulário de Lojas não está a conseguir buscar a lista de parceiros da API (`GET /api/parceiros`).
    * **Sua Tarefa:** Refatore o componente do formulário de Lojas para garantir que ele busque (`fetch`) e exiba corretamente a lista de parceiros no dropdown, permitindo que o administrador possa associar uma loja a um parceiro.

**Resumo da Missão:** Corrija estes dois problemas para que o módulo de Produtos volte a funcionar e para que seja possível criar e editar Lojas.
